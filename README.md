# XAI
UNSW COMP Research Thesis - z5210274

Commands for Game.py are:
- [WASD] to Move Enemy
- [IJKL] to Move Player
- Left + Right Arrow to Aim/Change Theta
- [1234] to change aim modes
- 0 to Autoshoot
- Space to Shoot
- R to Clear projectile cache
- P to Reset Player position
- O to Control Player
- M to Control Enemy
- T to Train TCAV
- X to Reload TCAV
- Z to Save TCAV
